# Pika Downloads Bot (Node.js Version)

A simple and stable Telegram file-sharing and ads bot built using Node.js + Telegraf.

## Features
✔ Admin login (chat ID + password)  
✔ File upload + auto URL generation  
✔ Users get file by start payload  
✔ Broadcast ads:  
   • /sendads – text ads  
   • /sendimgads – image ads  
   • /sendvidads – video ads  
✔ Stores all users automatically  
✔ 100% compatible with Render hosting
